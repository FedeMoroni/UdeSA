# UdeSA

Personal repository for practice and coursework.

## Contents
- Python
- C
- Java

## Structure
- `Python/`: scripts and Jupyter notebooks
- `C/`: C programming exercises
- `JavaUdesa/`: Java exercises

## Python Setup

This project uses a virtual environment (`.venv`).

### Activate environment
```bash
source .venv/bin/activate
```

### Install dependencies
```bash
pip install -r Python/requirements.txt
```

## Run Python example
```bash
python Python/helloWorld.py
```

## Run C example
```bash
gcc C/hola.c -o hola
./hola
```

## Run Java example
```bash
cd JavaUdesa
javac Main.java
java Main
```

## Notes
- `.venv` is not included in the repository
- This repository is for learning and experimentation
